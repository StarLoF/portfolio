O HTML, que é uma linguagem de marcação estrutura a página, em conjunto com o JavaScript, o qual torna as páginas HTML mais dinâmicas e interativas, são indispensáveis no desenvolvimento de frontend, tornando o desenvolvimento de páginas web interativa uma atividade mais fácil e palpável. Esta interação entre ambos permite, por exemplo, a criação de animações simples, até as mais complexas, aprimorando a experiência do usuário, assim como a usabilidade das páginas, podendo adicionar códigos que interagem com algum evento do usuário, assim como menus interativos, carrosséis, transições entre outros elementos.
